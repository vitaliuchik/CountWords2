Час робити програми:
2години 10хв. - багатопоточний варіант 
1 потік - зчитування файлу, 4 потоки - індексація-підрахунок слів, 4 потоки - об'єднування словників
розмір черг - 1000

Характеристики комп'ютера:
2 ядра (4 логічних), RAM 8GB, SSD 


Файл конфігурації передавати консольним аргументом (за замовчуванням config.dat)
python run.py configuration.txt 
або
python run.py


Скрипт сам білдить програму в в папці build і запускає її.
